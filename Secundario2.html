Windows 10 permitirá restaurar el sistema operativo desde la nube




La empresa Microsoft ha anunciado que el Windows 10 Insider Preview Build 18950 ya está disponible para los usuarios del anillo rápido.


Microsoft ha anunciado que está desarrollando una función denominada 'Cloud Download' que permitiría al usuario restaurar o refrescar un equipo con Windows 10.

La empresa Microsoft ha anunciado que el Windows 10 Insider Preview Build 18950 ya está disponible para los usuarios del anillo rápido, quienes podrán encontrar referencias a la nueva función 'Cloud Download' relacionada con restablecer el sistema operativo a través de la nube.

A pesar de ello, la herramienta aún no está disponible, ya que todavía está en la fase de desarrollo.

No obstante, esta función parece ser similar a Apple macOS Recovery, que permite recuperar el sistema operativo del equipo descargando una copia desde la nube, además de solicitar ayuda 'online' o reparar o eliminar un disco duro.
